Case Studies

Logicbroker ⇄ FlexShopper Custom Order/Lease & Accounting Systems

BA/PM · Integration · EDI & APIs

Led integration of Logicbroker with FlexShopper’s custom order/lease management systems and accounting databases (GCP & NetSuite). Delivered multi-vendor onboarding flows, custom architecture for non-standard vendors, and special invoice handling requirements. Produced BRD with system architecture diagram (Visio) and SOPs in Confluence for merchandising, accounting, and operations.

  • Impact: Cut vendor onboarding time by 50%, doubling capacity; reduced manual touches in accounting reconciliation.
  • Stack: Logicbroker (Azure, API-driven), NetSuite, GCP, EDI transactions, dashboards.
  • Artifacts: BRD, Visio system architecture, Confluence SOPs, Go-live Checklist.
  • Complexity: Supported vendors unable to onboard to Logicbroker; implemented custom invoice processing logic.

Custom Back-End Rebate System (UI + Logic)

BA · Product Design · Workshops & Requirements

Led BA efforts to design a custom rebate management application with a UI for the merchandising team. Facilitated cross-functional workshops with developers, designers, and merch to define use cases and required functionality. Addressed complex needs such as UI criteria setup for rebates, change control, SKU validation, and developer-facing error logging. Delivered user stories for all possible UI actions and example rebate criteria for both UI and back-end logic.

  • Impact: Project was deprioritized before launch; ensured visibility of work and status during business prioritization sessions.
  • Artifacts: User stories, rebate criteria documentation, functional specifications.
  • Complexity: Balancing merch UI needs with full-system functionality beyond rebate generation.
